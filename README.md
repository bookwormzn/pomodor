# Pomodor - Minimalist Pomodoro Zamanlayıcı 🍅

Bu proje, odaklanmanızı artırmak ve çalışma verimliliğinizi optimize etmek için tasarlanmış, web tabanlı, minimalist bir Pomodoro zamanlayıcısıdır. Pomodoro Tekniği'ni (25 dakika çalışma, 5 dakika mola) kolayca uygulamanıza yardımcı olur.

## 🚀 Özellikler

- **Basit Arayüz:** Göz yormayan, sade ve kullanıcı dostu tasarım.
- **Temel Kontroller:** Başlat, Duraklat ve Sıfırla fonksiyonları.
- **Otomatik Döngü:** Çalışma ve mola süreleri için önceden tanımlanmış ideal süreler.
- **Hafif Yapı:** Herhangi bir kurulum gerektirmez, doğrudan tarayıcı üzerinden çalışır.

## 🛠️ Kullanılan Teknolojiler

Bu proje tamamen web teknolojileri kullanılarak geliştirilmiştir:

- **HTML5:** Sayfa yapısı ve içeriği.
- **CSS3:** Modern ve temiz görünüm (index.html içinde gömülü).
- **JavaScript (Vanilla):** Zamanlayıcı mantığı ve kullanıcı etkileşimi (index.html içinde gömülü).

## 💻 Kurulum ve Kullanım

Proje herhangi bir bağımlılık veya sunucu yapılandırması gerektirmez. Kullanmak için:

1. Bu depoyu bilgisayarınıza indirin veya klonlayın:
   ```bash
   git clone https://github.com/bookwormzn/pomodor.git
   ```
2. Ana dizindeki `index.html` dosyasını herhangi bir modern web tarayıcısı (Chrome, Firefox, Safari vb.) ile açın.

## 📁 Dosya Yapısı

```text
pomodor/
└── index.html   # HTML, CSS ve JavaScript kodlarının tamamını içeren ana dosya.
```
